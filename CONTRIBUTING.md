# Contributing to HackinTune

First off, thank you for considering contributing to HackinTune! It's people like you who make the Hackintosh community great.

### Reporting Bugs & Doubts
*   Check the [Issues](https://github.com/sam4muzix/HackinTune/issues) to see if it's already documented.
*   If not, [open a new issue](https://github.com/sam4muzix/HackinTune/issues/new). Provide as much detail as possible.
*   For general doubts, feel free to [email shyam4muzix@gmail.com](mailto:shyam4muzix@gmail.com).

### Suggesting Enhancements
*   Open a new issue with a clear title and description of the proposed feature.

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure the build script runs successfully (`./build.sh`).
4.  Issue that pull request!
